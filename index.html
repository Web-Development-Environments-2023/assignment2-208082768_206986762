Welcome to Space Invaders Game!

Introduction

Your mission, should you choose to accept it, is to eliminate all the bad spaceships as fast as possible, and to protect yourself. Note that you only have 3 lives, make it count.
Good luck.


Rules

  • Use the arrow keys to move.
  • Use the spacebar to shoot
  • Note that there is a timer set to 2 mins, you can modify it through the settings screen.


Plugins
  • jQuery Validation Plugin.
